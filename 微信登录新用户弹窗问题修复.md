# 微信登录新用户弹窗问题修复

## 🎯 **问题描述**

用户反馈：**新用户没有弹出补充姓氏和性别的窗口**

## 🔍 **问题分析**

经过检查发现，问题出现在以下几个环节：

### **1. 后端问题**
- **AuthController异常处理不当** - 使用`ApiResponse.ok(response)`覆盖了BusinessException的状态码
- **缺少BusinessException导入** - 导致编译错误
- **使用了不存在的error方法** - ApiResponse只有fail方法

### **2. 前端问题**
- **request工具处理逻辑** - 只处理`code === 0`的情况，其他状态码都被reject
- **201状态码未正确传递** - 导致新用户无法触发补充信息弹窗

## 🛠️ **修复方案**

### **1. 后端修复**

#### **AuthController修改**
```java
// 添加BusinessException导入
import com.example.membersystem.common.BusinessException;

// 修改异常处理逻辑
@PostMapping("/wx-login")
public ApiResponse<LoginResponse> wxLogin(@RequestBody WxLoginRequest request) {
    try {
        LoginResponse response = authService.wxLogin(request);
        return ApiResponse.ok(response);
    } catch (BusinessException e) {
        System.out.println("微信登录业务异常: " + e.getMessage());
        System.out.println("异常状态码: " + e.getCode());
        
        if (e.getCode() == 201) {
            // 新用户需要补充信息
            return ApiResponse.fail(201, e.getMessage());
        } else {
            // 其他业务异常
            return ApiResponse.fail(e.getCode(), e.getMessage());
        }
    } catch (Exception e) {
        return ApiResponse.fail(500, "微信登录失败，请稍后重试");
    }
}
```

#### **AuthService逻辑保持**
```java
// 新用户检测逻辑
if (!StringUtils.hasText(request.getUserInfo().getLastName())) {
    throw new BusinessException(201, "新用户需要补充姓氏和性别信息");
}
```

### **2. 前端修复**

#### **request.js修改**
```javascript
success(res) {
    const body = res.data
    if (body && body.code === 0) {
        resolve(body.data)
        return
    }
    
    // 处理微信登录新用户需要补充信息的情况
    if (body && body.code === 201) {
        resolve(body)  // 直接返回完整的响应，让前端处理
        return
    }
    
    // 其他处理逻辑...
}
```

#### **profile.js处理逻辑保持**
```javascript
handleWxLoginResponse(response, tempWxInfo) {
    if (response.code === 200) {
        // 登录成功
        const userInfo = response.data
        wx.setStorageSync('userInfo', userInfo)
        wx.setStorageSync('token', response.token || '')
        this.checkLoginStatus()
    } else if (response.code === 201) {
        // 新用户，需要补充信息
        this.setData({
            showWxRegisterModal: true,
            tempWxInfo: tempWxInfo
        })
    } else {
        wx.showToast({
            title: response.message || '登录失败',
            icon: 'none'
        })
    }
}
```

## 📊 **技术细节**

### **状态码约定**
- **200** - 登录成功（老用户或新用户已补充信息）
- **201** - 新用户需要补充信息
- **其他** - 登录失败或错误

### **数据流设计**
```
前端请求 → request.js → 后端处理 → 响应返回 → request.js处理 → 前端逻辑
```

### **异常处理链**
```
AuthService → BusinessException → AuthController → ApiResponse.fail → request.js → 前端弹窗
```

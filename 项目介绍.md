# 会员管理系统

## 项目概述

本项目是一个面向美业门店（美睫、美容等）的**会员管理系统**，采用前后端分离架构，包含微信小程序客户端和Java Spring Boot后端服务。

---

## 技术架构

### 后端技术栈
- **框架**: Spring Boot 3.3.5
- **语言**: Java 17
- **数据库**: MySQL
- **ORM**: Spring Data JPA (Hibernate)
- **安全**: JWT (JSON Web Token) + Spring Security Crypto
- **密码加密**: BCrypt + Argon2 (混合加密)
- **API文档**: OpenAPI 3.0 (Swagger UI)
- **AI服务**: 腾讯元器AI (腾讯元宝)

### 前端技术栈
- **平台**: 微信小程序
- **语言**: JavaScript
- **样式**: WXSS (微信小程序样式表)

## 核心功能模块

### 1. 用户认证与授权 (auth)
- 短信验证码登录
- 密码登录
- 微信授权登录
- JWT Token认证
- 权限分级控制

### 2. 用户管理 (user)
- 用户信息管理
- 权限等级设置 (店长/员工/会员/游客)
- 用户状态管理 (启用/禁用)
- 手机号修改

### 3. 消费记录管理 (consume)
- 消费记录CRUD
- 按手机号/姓氏/日期范围查询
- 消费统计分析
- 充值金额统计

### 4. 美睫记录管理 (eyelash)
- 美睫服务记录管理
- 客户档案跟踪
- 服务历史查询
- 支持多条件组合搜索

### 5. 充值折扣管理 (discount)
- 充值折扣规则设置
- 满额折扣自动计算
- 折扣规则启用/禁用
- 按金额范围查询适用折扣

### 6. 轮播内容管理 (carousel)
- 首页视频展示
- 图片轮播
- 营销内容配置
- 内容动态更新

### 7. AI智能客服 (ai)
- 基于腾讯元器的智能对话
- 用户身份识别
- 快捷问题推荐
- 7x24小时在线服务

### 8. 系统统计 (stats)
- 会员总数统计
- 今日新增用户
- 活跃用户统计
- 权限分级数据展示

### 9. 安全模块 (security)
- 混合密码加密 (BCrypt + Argon2)
- 密码哈希性能优化
- 加密算法基准测试

## API接口概览

| 模块 | 基础路径 | 说明 |
|------|----------|------|
| 认证 | `/api/auth` | 登录、注册、验证码 |
| 用户 | `/api/users` | 用户CRUD、权限管理 |
| 消费记录 | `/api/consume-records` | 消费记录管理 |
| 美睫记录 | `/api/eyelash-records` | 美睫服务记录 |

| 轮播内容 | `/api/carousel` | 首页内容配置 |
| AI客服 | `/api/customer-service` | 智能客服接口 |
| 统计 | `/api/stats` | 系统统计数据 |

## 权限等级说明

| 等级 | 角色 | 权限描述 |
|------|------|----------|
| 1 | 店长 | 最高权限，可管理所有功能 |
| 2 | 员工 | 可查看成员列表、消费记录、统计数据 |
| 3 | 会员 | 可查看自己的消费记录 |
| 4 | 游客 | 仅可浏览首页内容 |

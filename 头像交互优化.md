# 头像交互优化

## 🎯 **优化目标**

修改个人信息弹窗的头像交互逻辑：
- **当前头像显示** - 显示用户当前头像或灰色圆形占位符
- **点击切换** - 点击后展开头像选择器
- **悬浮提示** - 鼠标悬浮时显示"点击更换头像"提示
- **动画效果** - 平滑的展开/收起动画

## 🛠️ **实现内容**

### **1. WXML结构优化**

#### **修改前**
```xml
<view class="avatar-selector">
  <view class="avatar-item" wx:for="{{avatarOptions}}" wx:key="index" 
        bindtap="selectAvatar" data-index="{{index}}">
    <image class="avatar-img" src="{{item}}" mode="aspectFill" />
    <view class="avatar-check">✓</view>
  </view>
</view>
```

#### **修改后**
```xml
<view class="current-avatar-container">
  <view class="current-avatar" bindtap="toggleAvatarSelector">
    <image wx:if="{{personalInfoForm.avatarUrl}}" 
           class="current-avatar-img" 
           src="{{personalInfoForm.avatarUrl}}" 
           mode="aspectFill" />
    <view wx:else class="current-avatar-placeholder">
      <text class="placeholder-icon">👤</text>
    </view>
    <view class="avatar-edit-hint">点击更换头像</view>
  </view>
</view>

<!-- 头像选择器 -->
<view class="avatar-options {{showAvatarOptions ? 'show' : ''}}" wx:if="{{showAvatarOptions}}">
  <view class="avatar-item" wx:for="{{avatarOptions}}" wx:key="index" 
        bindtap="selectAvatar" data-index="{{index}}">
    <image class="avatar-img {{personalInfoForm.avatarUrl === item ? 'selected' : ''}}" 
           src="{{item}}" mode="aspectFill" />
    <view class="avatar-check" wx:if="{{personalInfoForm.avatarUrl === item}}">✓</view>
  </view>
</view>
```

### **2. JavaScript逻辑优化**

#### **数据字段添加**
```javascript
data: {
  showAvatarOptions: false,     // 头像选择器显示状态
  // 其他数据...
}
```

#### **方法添加**
```javascript
// 切换头像选择器显示状态
toggleAvatarSelector() {
  this.setData({
    showAvatarOptions: !this.data.showAvatarOptions
  })
},

// 选择头像（优化后）
selectAvatar(e) {
  const index = e.currentTarget.dataset.index
  const { avatarOptions } = this.data
  const selectedAvatar = avatarOptions[index]
  
  this.setData({
    'personalInfoForm.avatarUrl': selectedAvatar,
    showAvatarOptions: false  // 选择后自动关闭选择器
  })
},

// 隐藏弹窗时重置状态
hidePersonalInfoModal() {
  this.setData({
    showPersonalInfoModal: false,
    showAvatarOptions: false  // 关闭弹窗时重置头像选择器状态
  })
}
```

## 📊 **交互流程**

### **用户操作流程**
```
1. 用户看到当前头像（或灰色占位符）
     ↓
2. 鼠标悬浮时显示"点击更换头像"提示
     ↓
3. 点击头像展开选择器（动画效果）
     ↓
4. 选择新头像（自动关闭选择器）
     ↓
5. 当前头像更新为新选择的头像
```

### **状态管理**
```
showAvatarOptions: false → 点击头像 → showAvatarOptions: true
     ↓
选择新头像 → showAvatarOptions: false（自动关闭）
     ↓
关闭弹窗 → showAvatarOptions: false（重置状态）
```

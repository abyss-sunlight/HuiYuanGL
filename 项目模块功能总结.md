# 会员管理系统 - 模块功能总结

## 项目概述

这是一个基于微信小程序 + Spring Boot的会员管理系统，专门为美容美睫店设计。系统采用前后端分离架构，提供完整的会员管理、消费记录、睫毛记录等功能。

---

## 技术架构

### 后端技术栈
- **框架**: Spring Boot 3.3.5
- **数据库**: MySQL 8.0 (HuiYuanSql)
- **ORM**: Spring Data JPA + Hibernate
- **认证**: JWT + Spring Security
- **文档**: Swagger/OpenAPI 3
- **AI集成**: 腾讯元器AI服务
- **密码加密**: BCrypt + 自定义密码编码器配置

### 前端技术栈
- **平台**: 微信小程序
- **架构**: 模块化JavaScript
- **UI**: 原生小程序组件 + 自定义样式

## 核心模块详解

### 1. 用户认证模块 (auth)

**功能职责**: 处理用户登录、注册、权限验证

**主要功能**:
- **多种登录方式**:
  - 短信验证码登录
  - 密码登录
  - 微信授权登录
- **账户管理**:
  - 设置/重置密码
  - 修改手机号（双重验证）
  - 发送短信验证码
- **权限控制**:
  - JWT令牌生成与验证
  - 基于注解的权限控制
  - 拦截器统一鉴权

**权限等级**:
1. 店长 - 最高权限
2. 员工 - 管理权限
3. 会员 - 普通权限
4. 游客 - 基础权限

**关键文件**:
- `AuthController.java` - 认证接口控制器
- `AuthService.java` - 认证业务逻辑
- `JwtUtil.java` - JWT工具类
- `AuthInterceptor.java` - 认证拦截器

---

### 2. 用户管理模块 (user)

**功能职责**: 管理系统用户信息

**主要功能**:
- **用户信息管理**:
  - 用户基本信息增删改查
  - 头像上传与管理
  - 会员号自动生成
- **权限管理**:
  - 用户权限等级设置
  - 账户状态管理（正常/禁用）
- **余额管理**:
  - 账户余额查询
  - 折扣率设置

**数据字段**:
- 基本信息：手机号、姓名、性别、头像
- 业务信息：会员号、余额、折扣率
- 系统信息：权限等级、账户状态

**关键文件**:
- `UserController.java` - 用户管理接口
- `User.java` - 用户实体类

---

### 3. 睫毛记录模块 (eyelash)

**功能职责**: 记录会员的睫毛护理和嫁接服务

**主要功能**:
- **记录管理**:
  - 睫毛服务记录创建
  - 历史记录查询
  - 记录修改与删除
- **服务信息**:
  - 款式选择（自然款、浓密款、猫眼款等）
  - 型号管理
  - 长度记录（毫米单位）
  - 翘度设置（C、D、J等）
- **数据统计**:
  - 按手机号查询记录
  - 按日期范围筛选
  - 服务频次分析

**数据字段**:
- 客户信息：手机号、姓名、性别
- 服务信息：款式、型号、长度、翘度
- 时间信息：记录日期、创建/更新时间

**关键文件**:
- `EyelashRecordController.java` - 睫毛记录接口
- `EyelashRecord.java` - 睫毛记录实体

---

### 4. 消费记录模块 (consume)

**功能职责**: 管理会员的消费和充值记录

**主要功能**:
- **消费管理**:

**数据字段**:
- 会员信息：手机号、姓名、性别
- 消费信息：消费金额、消费项目、消费类型

**关键文件**:
- `ConsumeRecordController.java` - 消费记录接口
- `ConsumeRecord.java` - 消费记录实体

### 5. 充值折扣管理模块 (discount)

**功能职责**: 管理充值活动和折扣策略

**主要功能**:

- 统计信息：使用次数、效果统计

**关键文件**:
- `RechargeDiscountController.java` - 充值折扣接口
- `RechargeDiscount.java` - 折扣实体

## 数据库设计

### 核心数据表

1. **user** - 用户信息表
   - 存储用户基本信息、权限、余额等
   - 主键：id (自增)
   - 唯一索引：phone, openid, member_no

2. **eyelash_records** - 睫毛记录表
   - 存储睫毛服务记录
   - 关联字段：phone (用户手机号)

3. **consume_records** - 消费记录表
   - 存储消费和充值记录
   - 关联字段：phone (用户手机号)

### 4. recharge_discount - 充值折扣表
   - 存储充值活动和折扣配置
   - 支持金额档位和折扣率设置
   - 主键：id (自增)
   - 索引：amount, is_active

5. **carousel_content** - 轮播内容表
   - 存储首页轮播内容
   - 支持视频、图片、文字内容
   - 主键：id (自增)

6. **系统配置表** (新增)
   - 存储系统配置参数
   - 支持动态配置管理
   - 主键：id (自增)

## 总结

该会员管理系统是一个功能完整、架构清晰、技术先进的美容美睫店管理解决方案。系统具备以下特点：

1. **功能全面**: 涵盖会员管理、服务记录、消费管理、AI客服、充值折扣、数据统计等核心业务
2. **权限分级**: 四级权限体系，满足不同角色需求，精细化权限控制
3. **技术先进**: 采用现代化技术栈，支持高并发和大数据量，集成AI智能服务
4. **用户体验**: 微信小程序原生体验，操作简洁直观，AI客服悬浮窗设计
5. **智能化**: 集成AI客服和数据分析，提供智能问答和业务洞察
6. **可扩展**: 模块化设计，便于后续功能扩展和维护
7. **安全可靠**: BCrypt密码加密、JWT认证、时序攻击防护等多重安全保障

**新增亮点功能**:
- **AI智能客服**: 基于腾讯元器AI的24小时智能客服，支持页面跳转和个性化服务
- **AI数据分析**: 店长专属的智能数据分析功能，提供业务洞察和优化建议
- **充值折扣管理**: 灵活的充值折扣配置系统，支持多种优惠策略
- **数据统计分析**: 完善的会员统计和业务数据监控功能
- **密码安全增强**: 自定义密码编码器配置，支持BCrypt混合优化

系统已具备商业化运营的基础条件，可直接用于美容美睫店的日常经营管理，并通过AI智能化功能显著提升服务效率和用户体验。

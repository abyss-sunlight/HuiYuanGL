# 页面路径映射与客服问答

## A. 页面路径映射

### 核心页面路径表

| 用户称呼 | 小程序路径 | 页面功能 | 访问权限 |
| :--- | :--- | :--- | :--- |
| **会员中心/我的主页** | `pages/profile/profile` | 查看余额、等级、个人信息 | 会员及以上 |
| **消费记录页** | `pages/records/records?type=consume` | 查看消费历史和账单 | 会员及以上 |
| **美睫记录页** | `pages/records/records?type=eyelash` | 查看美睫服务历史 | 所有用户 |
| **充值折扣介绍页** | `pages/discount-management/discount-management` | 查看充值优惠信息 | 员工及以上 |
| **设置密码页** | `pages/profile/profile?action=setPassword` | 设置登录密码 | 所有用户 |
| **修改手机号页** | `pages/profile/profile?action=changePhone` | 修改绑定手机号 | 所有用户 |
| **联系店长页** | 通过客服电话/微信联系 | 获取店长联系方式 | 所有用户 |
| **会员管理页** | `pages/members/members` | 会员信息管理 | 员工及以上 |
| **AI分析页** | `pages/ai-analysis/ai-analysis` | 数据分析和报表 | 店长专用 |
| **首页** | `pages/index/index` | 系统首页和导航 | 所有用户 |

### 页面访问说明

**底部导航栏页面**
- **主页**：系统首页，显示基本信息和快捷入口
- **记录**：包含消费记录和美睫记录的查看功能
- **我的**：个人中心，显示余额、权限等级等

**权限控制页面**
- **充值折扣页**：只有员工和店长可以查看
- **会员管理页**：只有员工和店长可以访问
- **AI分析页**：仅限店长使用

### 页面跳转逻辑

```
首页 → 记录页面 → 选择记录类型 → 查看详情
首页 → 我的页面 → 查看个人信息 → 联系客服

带参数跳转示例：
- 跳转消费记录：pages/records/records?type=consume
- 跳转美睫记录：pages/records/records?type=eyelash
- 设置密码弹窗：pages/profile/profile?action=setPassword
- 修改手机号弹窗：pages/profile/profile?action=changePhone
- 日期筛选记录：pages/records/records?type=eyelash&startDate=2024-03-20&endDate=2024-03-23
```

### Q6: 我想设置密码/修改手机号，怎么操作？

**店长回答：**
您好！设置密码和修改手机号很方便的，我来教您操作：

**设置密码步骤：**
1. **打开小程序**：用您的手机号登录会员管理系统
2. **点击"我的"**：在底部导航栏找到"我的"选项
3. **进入个人中心**：点击头像或个人信息区域

6. **确认密码**：再次输入密码确认
7. **完成设置**：密码设置成功，下次可用密码登录

**修改手机号步骤：**
1. **打开小程序**：登录会员管理系统
2. **点击"我的"**：进入个人中心
3. **修改手机号**：系统会自动弹出"修改手机号"对话框
4. **验证身份**：输入当前密码验证身份
5. **输入新手机号**：输入您要更换的新手机号码
6. **获取验证码**：点击获取短信验证码
7. **输入验证码**：输入收到的6位验证码
8. **完成修改**：手机号修改成功

**安全提醒：**
- 密码建议使用6位数字，便于记忆
- 不要使用过于简单的密码如123456

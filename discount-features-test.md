# 充值折扣管理功能测试清单

## ✅ 已实现功能

### 1. 启用/禁用切换
- **UI组件**: Switch滑动开关
- **API接口**: `PUT /api/recharge-discount/{id}/status`
- **本地更新**: 状态切换后立即更新界面
- **错误处理**: 失败时自动恢复原状态

### 2. 编辑功能
- **触发方式**: 点击"编辑"按钮
- **表单预填充**: 自动填充当前数据
- **API接口**: `PUT /api/recharge-discount/{id}`
- **本地更新**: 编辑成功后更新对应项
- **标题显示**: "编辑折扣规则"

### 3. 删除功能
- **触发方式**: 点击"删除"按钮
- **确认对话**: 显示确认弹窗
- **API接口**: `DELETE /api/recharge-discount/{id}`
- **本地更新**: 从列表中移除对应项
- **安全提示**: 显示具体金额信息

### 4. 添加功能
- **触发方式**: 点击"添加折扣规则"按钮
- **表单验证**: 必填字段验证
- **API接口**: `POST /api/recharge-discount`
- **本地更新**: 添加到列表末尾
- **默认值**: 自动设置当前日期

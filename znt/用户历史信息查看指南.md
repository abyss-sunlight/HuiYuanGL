# 用户历史信息查看指南

欢迎使用我们的会员管理系统！作为会员，您可以随时查看自己的消费记录和美睫服务历史。下面为您详细介绍如何查看这些信息。

---

## A. 消费记录查看指南

### 什么是消费记录？

消费记录是您在本店所有资金往来的详细记录，包括每次充值和消费的完整信息。就像您的个人账本一样，让您清楚了解每一笔钱的去向。

### 一条消费记录包含哪些信息？

每条消费记录都会显示以下详细信息：

| 信息项 | 说明 | 示例 |
| :--- | :--- | :--- |
| **消费日期** | 交易发生的具体日期 | 2024-01-15 |
| **消费项目** | 具体的服务或充值项目 | 睫毛护理、会员充值 |
| **消费金额** | 这笔交易的金额 | 200.00元 |
| **消费类型** | 是"充值"还是"支出" | 支出、充值 |
| **账户余额** | 交易后的剩余金额 | 1000.00元 |
| **记录时间** | 系统记录的时间 | 2024-01-15 14:30:00 |

### 消费记录类型说明

**1. 充值记录**
- 显示您每次充款的金额和时间
- 充值后余额会增加
- 可能会显示享受的折扣优惠

**2. 支出记录**
- 显示您每次消费服务的费用
- 消费后余额会相应减少
- 详细列出消费的具体项目

### 如何查看消费记录？

在小程序中查看消费记录的步骤：

1. **打开小程序**：使用手机号登录会员管理系统
2. **进入"记录"页面**：点击底部导航栏的"记录"选项
3. **系统自动跳转**：页面会直接跳转到消费记录页面
4. **查看详情**：滑动查看所有历史记录，点击单条记录可查看详细信息
   - **注意**：消费记录和美睫记录在同一个页面，通过顶部切换按钮选择类型

### 消费记录的作用

- **账目透明**：清楚了解每次消费的具体情况
- **余额管理**：实时掌握账户余额变化
- **消费分析**：了解自己的消费习惯和频率
- **问题核对**：如有疑问可核对具体交易记录

## B. 美睫记录查看指南

### 什么是美睫记录？

美睫记录是您每次美睫服务的详细档案，记录了睫毛的款式、长度、翘度等专业参数。就像您的美睫日记，帮助您找到最适合的睫毛款式。

### 一条美睫记录包含哪些信息？

每条美睫记录都会保存以下专业信息：

| 信息项 | 说明 | 示例 |
| :--- | :--- | :--- |
| **服务日期** | 美睫服务的具体日期 | 2024-01-15 |
| **睫毛款式** | 美睫的整体风格类型 | 自然款、浓密款、猫眼款 |
| **款式型号** | 款式的具体编号 | 001、002、003 |
| **睫毛长度** | 睫毛的实际长度（毫米） | 10.5mm、12.0mm |
| **睫毛翘度** | 睫毛的卷翘程度 | C翘、D翘、J翘 |
| **记录时间** | 系统保存记录的时间 | 2024-01-15 14:30:00 |

### 美睫参数通俗解释

**1. 睫毛款式**
- **自然款**：看起来自然真实，适合日常妆容
- **浓密款**：睫毛比较浓密，效果明显
- **猫眼款**：眼尾处加长，有猫眼效果
- **仙女款**：仙气飘飘，比较梦幻
- **韩式款**：韩式风格，自然又有层次

**2. 睫毛长度**
- 单位是毫米（mm）
- 9-10mm：自然短款
- 11-12mm：中等长度
- 13mm以上：较长效果

**3. 睫毛翘度**
- **C翘**：自然翘度，适合日常
- **D翘**：比较卷翘，效果明显
- **J翘**：轻微上翘，很自然

### 如何查看美睫记录？

在小程序中查看美睫记录的步骤：

1. **打开小程序**：使用手机号登录会员管理系统
2. **进入"记录"页面**：点击底部导航栏的"记录"选项
3. **系统自动跳转**：页面会直接跳转到美睫记录页面
4. **查看历史记录**：滑动查看所有历史美睫服务记录
5. **对比效果**：可以对比不同时期的服务效果
   - **注意**：消费记录和美睫记录在同一个页面，通过顶部切换按钮选择类型

### 按时间筛选美睫记录

如果您想查看特定时间段的美睫记录，可以使用日期筛选功能：

1. **进入美睫记录页面**：按上述步骤进入记录页面
2. **点击日期筛选**：在页面顶部找到"日期筛选"按钮
3. **设置时间范围**：
   - 点击"开始日期"，选择起始日期
   - 点击"结束日期"，选择结束日期
4. **确认筛选**：点击"确定"按钮
5. **查看结果**：页面会显示指定时间段内的所有美睫记录

**常用时间范围示例**：
- **最近3天**：从3天前到今天
- **最近一周**：从7天前到今天
- **最近一个月**：从30天前到今天

### 美睫记录的作用

- **效果对比**：对比不同款式的实际效果
- **找到最爱**：记录最适合您的睫毛参数
- **服务连续**：保持美睫效果的连续性
- **专业建议**：为下次服务提供参考依据
- **效果追踪**：追踪美睫效果的持续时间

## C. 常见问题解答

### Q1: 我可以查看多久以前的记录？
**A**: 系统会保存您所有的历史记录，从您成为的游客第一天开始，都可以查看。

### Q2: 如果发现记录有错误怎么办？
**A**: 请及时联系店长或客服，我们会核实并为您更正记录信息。

### Q3: 美睫记录中的参数我不太懂怎么办？
**A**: 可以咨询我们的美睫师，她们会为您详细解释各项参数的含义和效果。

### Q4: 消费记录和实际消费不符怎么办？
**A**: 请保留相关凭证，联系店长进行核对，我们会尽快为您处理。

### Q5: 可以导出或打印我的记录吗？
**A**: 目前系统支持在线查看，如需导出记录请联系店长协助处理。

### Q6: 如何设置登录密码？
**A**: 设置密码很简单，按以下步骤操作：
1. 打开小程序，用手机号登录
2. 点击底部"我的"进入个人中心
3. 点击头像或个人信息区域
4. 系统会自动弹出"设置密码"对话框
5. 输入6位数字密码并确认
6. 设置成功，下次可用密码登录

### Q7: 如何修改绑定的手机号？
**A**: 修改手机号的步骤如下：
1. 打开小程序，登录会员管理系统
2. 点击底部"我的"进入个人中心
3. 系统会自动弹出"修改手机号"对话框
4. 输入当前密码验证身份
5. 输入新手机号并获取验证码
6. 输入短信验证码确认
7. 修改成功，可用新手机号登录

### Q8: 忘记密码怎么办？
**A**: 如果忘记密码，请联系店长重置密码。店长电话：18726685085。

# 美甲美睫门店数据库结构说明

## 核心数据表

### 表：user (用户表)
| 字段名 | 类型 | 说明 | 示例 |
| :--- | :--- | :--- | :--- |
| id | BIGINT | 用户ID，主键自增 | 1 |
| username | VARCHAR(64) | 用户名，默认手机号后4位 | 8001 |
| last_name | VARCHAR(50) | 姓氏，必填 | 张 |
| phone | VARCHAR(32) | 手机号，唯一标识，可用于登录 | 13800138001 |
| password | VARCHAR(128) | 密码（BCrypt哈希），可为空（纯短信登录） | $2a$10$N.zmdr9k7uOCQb376NoUnuTJ8iAt6Z5EHsM8lE9lBOsl7iKTVEFDa |
| openid | VARCHAR(128) | 微信openid，可为空 | o1234567890abcdef |
| gender | TINYINT | 性别：1-男，2-女，必填 | 1 |
| member_no | VARCHAR(20) | 会员号（唯一值），格式：年月日时分秒+2位随机数 | 202401011200001 |
| amount | DECIMAL(10,2) | 账户金额，可为空 | 1000.00 |
| discount | DECIMAL(5,2) | 折扣率，可为空 | 0.95 |
| permission_level | INT | 权限等级：1-店长，2-员工，3-会员，4-游客 | 3 |
| status | INT | 账户状态：0-正常，1-禁用 | 0 |
| avatar_url | VARCHAR(512) | 头像URL | https://example.com/avatar.jpg |
| created_at | DATETIME | 创建时间 | 2024-01-01 12:00:00 |
| updated_at | DATETIME | 更新时间 | 2024-01-01 12:00:00 |

**权限等级说明**：
- 1: 店长 - 拥有所有权限
- 2: 员工 - 拥有部分管理权限
- 3: 会员 - 普通用户权限
- 4: 游客 - 最低权限

| 字段名 | 类型 | 说明 | 示例 |
| :--- | :--- | :--- | :--- |

| is_active | TINYINT(1) | 是否启用：1-启用，0-禁用 | 1 |
| created_by | VARCHAR(50) | 创建人 | admin |
| created_at | TIMESTAMP | 创建时间 | 2024-01-15 14:30:00 |
| updated_at | TIMESTAMP | 更新时间 | 2024-01-15 14:30:00 |

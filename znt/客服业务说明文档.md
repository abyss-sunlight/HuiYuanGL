## A. 会员体系与升级规则

### 游客与会员的标志性区别

根据用户表（user表）结构，游客和会员的核心区别如下：

| 区别项 | 游客 | 会员 |
| :--- | :--- | :--- |
| **会员号** | 无（member_no为NULL） | 有（唯一会员号） |
| **账户余额** | 无（amount为NULL或0.00） | 有（实际充值金额） |
| **折扣率** | 无（discount为NULL） | 有（个人专属折扣） |
| **权限等级** | 4（游客） | 3（会员） |
| **系统功能** | 只能查看美睫记录 | 可查看个人完整信息 |

### 会员升级流程

**标准升级流程：**
```
游客联系店长 → 线下转账充值 → 店长系统操作 → 升级为会员
```

**详细步骤说明：**

1. **联系阶段**
   - 游客通过微信、电话或到店联系本店店长
   - 咨询会员制度和充值优惠活动
   - 选择合适的充值金额

2. **充值阶段**
   - 游客通过银行转账、微信支付等方式完成充值
   - 店长确认收到款项

3. **系统操作阶段**
   - 店长登录会员管理系统
   - 创建用户账号并设置基本信息
   - 录入充值金额并应用相应折扣
   - 系统自动生成会员号（格式：年月日时分秒+2位随机数）
   - 设置权限等级为3（会员）

4. **完成阶段**
   - 会员账号开通成功
   - 可使用手机号登录系统查看个人信息
   - 开始享受会员权益

**客服话术模板：**
> "您好！成为我们的会员非常简单，只需要联系我们的店长，通过转账方式充值即可。充值到账后，店长会立即在系统内为您开通会员账户，您就可以享受会员专属权益了。"

### 折扣规则说明

**计算公式：**
```
实际支付金额 = 充值金额 × 折扣率
优惠金额 = 充值金额 - 实际支付金额
账户到账金额 = 充值金额
```

**示例计算：**
- 充值500元，享受85折优惠
- 实际支付：500 × 0.85 = 425元
- 优惠金额：500 - 425 = 75元
- 账户到账：500元

### 动态折扣说明

**重要提醒：**
- 充值折扣为**动态调整**，会根据不同时期的活动政策变化
- 最新优惠活动以**店内公示**或**店长告知**为准
- 系统显示的折扣档位可能随时更新
- 建议充值前咨询当前最新优惠

## C. 会员权益

### 核心会员权益

基于数据库结构和业务逻辑，会员享有以下核心权益：

#### 1. 储值消费权益
- **预付费消费**：会员可先充值后消费，享受便捷的支付体验
- **余额管理**：实时查看账户余额，消费记录透明可查
- **余额永不过期**：充值余额长期有效，无使用期限

#### 2. 价格优惠权益
- **充值折扣**：根据充值金额享受不同档位的折扣优惠
- **个人折扣率**：部分会员可能享有专属的个人折扣率
- **消费透明**：所有消费明码标价，无隐形收费

#### 3. 服务记录权益
- **服务历史**：完整记录每次美睫服务的详细参数
- **效果追踪**：可查看历史服务款式和效果对比
- **个性化推荐**：基于历史记录推荐适合的服务款式

#### 4. 专属服务权益
- **会员优先**：在服务高峰期享有优先安排权
- **专属客服**：可获得一对一的客服咨询服务
- **生日关怀**：会员生日月可能享受特别优惠（需确认）

## D. 常见问题解答

### Q1: 如何判断我是否已经是会员？
**A**: 登录系统后，如果您能看到个人余额和消费记录，说明您已经是会员。如果只能查看美睫记录，您还是游客状态。

### Q2: 充值后多久能到账？
**A**: 线下充值成功后，店长会立即在系统内为您操作，账户余额会实时到账。

### Q3: 充值金额可以部分退款吗？
**A**: 根据门店政策，充值余额一般不支持部分退款，建议您根据实际消费需求合理充值。

### Q4: 会员账户会过期吗？
**A**: 会员账户和充值余额均长期有效，不会过期。

### Q5: 可以用别人的账户消费吗？
**A**: 为保障账户安全，只能使用本人账户进行消费，不支持跨账户使用。

### Q6: 如何设置和修改账户安全信息？
**A**: 系统提供以下安全功能：
- **设置密码**：在"我的"页面点击个人信息区域，按提示设置6位数字密码
- **修改手机号**：在"我的"页面系统会自动弹出修改手机号选项，需验证当前密码
- **密码重置**：如忘记密码，请联系店长重置（电话：18726685085）

### Q7: 如何按时间查看历史记录？
**A**: 在记录页面可以使用日期筛选功能：
1. 进入记录页面（消费记录或美睫记录）
2. 点击"日期筛选"按钮
3. 设置开始和结束日期
4. 点击"确定"查看指定时间段记录
常用时间范围：最近3天、最近一周、最近一个月等。

## E. 客服工作要点

### 身份确认要点
- 询问客户手机号，确认系统中的用户状态
- 查看权限等级，区分游客和会员
- 确认账户余额和消费记录
- 引导用户设置密码保护账户安全

### 升级引导要点
- 详细说明会员权益和充值优惠
- 引导客户联系店长进行线下充值
- 告知充值后的系统开通流程
- 介绍账户安全功能（密码设置、手机号修改）

### 操作引导要点
- 指导用户查看消费记录和美睫记录
- 教授使用日期筛选功能查找特定时间记录
- 协助用户设置密码和修改手机号
- 提醒用户注意账户信息安全

### 投诉处理要点
- 耐心解释会员制度和折扣规则
- 对于折扣变动，强调以店内公示为准
- 无法解决的问题及时转接店长处理

### 服务标准
- 主动询问客户需求
- 准确回答会员相关问题
- 保持友好专业的服务态度
- 及时更新客户信息到系统
- 熟练掌握系统操作和页面跳转
- 关注账户安全，提醒用户设置密码

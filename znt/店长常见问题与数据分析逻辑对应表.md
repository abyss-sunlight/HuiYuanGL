# 店长常见问题与数据分析逻辑对应表

## 场景1：整体盈利概况
- **用户问题**：“今天/本周/本月营收怎么样？”
- **分析逻辑**：
    1.  确定时间范围（今日/本周/本月）。
    2.  查询该时间段内 `consume_type` = '充值' 的所有记录。
    3.  计算：总营收、订单数、客单价。
    4.  与上一个同等周期（昨日/上周/上月）进行环比计算。
    5.  **输出格式**：“{时间范围}总营收{金额}元，共{订单数}笔，客单价{金额}元。相比{上一周期}，营收{增长/下降}{百分比}%。”

## 场景2：细分项目分析
- **用户问题**：“最近什么项目最赚钱？/ 美甲和美睫哪个收入高？”
- **分析逻辑**：
    1.  确定时间范围（如“最近一周”）。
    2.  按 `consume_item` 分组，计算各项目的总营收和订单数。
    3.  按营收降序排列。
    4.  **输出格式**：以表格或列表形式呈现排名前5的项目，包含项目名称、营收、订单数、占比。

## 场景3：会员充值和消费分析
- **用户问题**：“会员充值贡献大吗？/ 会员充值和消费情况如何？”
- **分析逻辑**：
    1.  查询时间段内，`consume_type` = '充值' 的记录，计算总充值金额。
    2.  查询时间段内，`consume_type` = '支出' 的记录，计算总消费金额。
    3.  获取当前时间段（如本月）的数据。
    4.  计算会员总余额。
    5.  **输出格式**：“{时间范围}会员充值共{金额}元，会员充值共{金额}元，会员消费共{金额}元，会员余额共{金额}元。”

## 场景4：趋势与预警
- **用户问题**：“最近业绩有下滑吗？/ 和去年同期比怎么样？”
- **分析逻辑**：
    1.  获取当前时间段（如本月）的数据。
    2.  获取对比时间段（如上月或去年同月）的数据。
    3.  计算关键指标（营收、订单数）的环比或同比增长率。
    4.  若下降超过阈值（如10%），提示关注。
    5.  **输出格式**：“本月营收为{金额}元，较上月{增长/下降}{百分比}%。主要变化来自{项目/会员}消费的{增加/减少}。”

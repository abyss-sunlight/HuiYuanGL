# 个人中心菜单优化

## 🎯 **优化目标**

移除"我的"页面中的"个人信息"、"设置"、"帮助与反馈"菜单项，简化用户界面。

## 📋 **修改内容**

### **移除的菜单项**
1. **个人信息** - 查看用户详细信息
2. **设置** - 应用设置选项
3. **帮助与反馈** - 帮助信息和反馈功能

### **保留的菜单项**
1. **设置密码** - 设置/重置登录密码
2. **关于** - 应用关于信息

## 🛠️ **具体修改**

### **2. JavaScript文件修改**

#### **移除的函数**
```javascript
// 移除的函数（如果存在）
viewUserInfo() {
  // 查看个人信息功能
}

viewSettings() {
  // 设置功能
}

viewHelp() {
  wx.showToast({
    title: '帮助功能开发中',
    icon: 'none'
  })
}
```

#### **保留的函数**
```javascript
// 保留的函数
goToSetPassword() {
  wx.navigateTo({
    url: '/pages/set-password/set-password'
  })
}

viewAbout() {
  wx.showModal({
    title: '关于',
    content: '会员管理系统 v1.0\n\n基于 Spring Boot + 微信小程序开发\n提供完整的用户认证和会员管理功能',
    showCancel: false
  })
}
```

充值折扣管理测试

问题检查清单：

  • ✅ 页面已在 app.json 中注册
  • ✅ 后端实体类使用 javax.persistence
  • ✅ BigDecimal 舍入模式已修正
  • ✅ 前端页面文件已创建
  • ✅ 后端API已创建
  • ✅ 数据库表结构已定义

可能的问题：

  1. 数据库表未创建 - 需要执行 40_recharge_discount.sql
  2. 后端服务未启动 - 检查Spring Boot应用状态
  3. 权限问题 - 确保用户是店长权限（permissionLevel = 1）
  4. 网络连接 - 检查小程序是否正确连接到后端
  5. 登录状态 - 确保用户已登录并存储了用户信息

调试步骤：

  1. 打开小程序开发者工具控制台
  2. 点击"充值折扣"按钮
  3. 查看控制台输出的调试信息
  4. 检查是否有错误信息
  5. 验证用户权限级别是否为1（店长）

数据库执行命令：

-- 在MySQL中执行
USE HuiYuanSql;
SOURCE /path/to/40_recharge_discount.sql;

-- 或者手动执行
CREATE TABLE IF NOT EXISTS recharge_discount (
  id BIGINT PRIMARY KEY AUTO_INCREMENT COMMENT '折扣ID',
  recharge_amount DECIMAL(10,2) NOT NULL COMMENT '充值金额',
  discount_rate DECIMAL(5,4) NOT NULL COMMENT '折扣率（0.0000-1.0000）',
  discount_percentage DECIMAL(5,2) NOT NULL COMMENT '折扣百分比（0.00-100.00）',
  effective_date DATE NOT NULL COMMENT '生效日期',
  is_active TINYINT(1) NOT NULL DEFAULT 1 COMMENT '是否启用：1-启用，0-禁用',
  created_by VARCHAR(50) NOT NULL COMMENT '创建人',
  created_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP COMMENT '创建时间',
  updated_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP ON UPDATE CURRENT_TIMESTAMP COMMENT '更新时间'
) COMMENT='充值折扣表';

# 项目核心代码文件清单

## 项目概述

这是一个基于Spring Boot + 微信小程序的会员管理系统，包含用户管理、消费记录、AI分析、优惠管理等功能模块。

## 技术栈

### 后端技术栈
- **Spring Boot 3.3.5** - 主框架
- **Spring Security** - 安全框架
- **Spring Data JPA** - 数据访问层
- **MySQL** - 数据库
- **JWT** - 认证令牌
- **OpenAPI/Swagger** - API文档
- **Lombok** - 代码简化

### 前端技术栈
- **微信小程序原生开发** - 前端框架
- **JavaScript ES6+** - 编程语言
- **WXML** - 页面结构
- **WXSS** - 页面样式
- **微信小程序API** - 原生API

---

## 项目特色功能

1. **多端登录支持** - 支持密码登录、短信登录、微信登录
2. **会员等级管理** - 完整的会员等级和积分体系
3. **消费记录追踪** - 详细的消费和美睫记录管理
4. **AI智能分析** - 基于AI的数据分析和客服功能
5. **优惠营销系统** - 灵活的优惠券和充值优惠管理
6. **轮播图管理** - 动态内容展示功能

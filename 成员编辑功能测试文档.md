# 成员编辑功能测试文档

## 功能概述
本文档用于测试成员列表编辑功能的完整实现，包括前端弹窗、状态切换和后端API接口。

## 测试环境
- 前端：微信小程序
- 后端：Spring Boot API
- 数据库：MySQL

## 测试用例

### 用例1：正常状态切换
**步骤：**
1. 使用店长账户登录
2. 进入成员列表页面
3. 点击某个员工的编辑按钮
4. 在弹窗中切换状态滑动块
5. 点击确认

**预期结果：**
- 弹窗显示正确用户信息
- 状态切换成功
- 显示成功提示消息
- 后端记录状态变更日志

### 用例2：权限验证
**步骤：**
1. 使用员工账户登录
2. 尝试修改店长状态

**预期结果：**
- 返回权限不足错误
- 状态不会被修改

### 用例3：参数验证
**步骤：**
1. 发送无效状态值到API

**预期结果：**
- 返回参数验证错误
- 状态不会被修改

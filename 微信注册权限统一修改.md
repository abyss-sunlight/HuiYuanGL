# 微信注册权限统一修改

## 🎯 **修改目标**

让微信注册的新用户和短信注册的新用户一样，权限为游客（权限级别4）。

## 🔍 **权限体系分析**

### **权限等级定义**
```
1 - 店长：拥有所有权限，系统管理员
2 - 员工：拥有部分管理权限，操作员
3 - 会员：普通用户权限，可查看和操作自己的数据
4 - 游客：最低权限，只能进行基本查看
```

### **短信注册新用户权限**
```java
// 短信注册新用户默认权限
user.setPermissionLevel(4); // 默认权限为游客
```

### **微信注册新用户权限（修改前）**
```java
// 微信注册新用户默认权限
user.setPermissionLevel(3); // 默认为会员权限
```

## 🛠️ **修改内容**

### **修改微信注册权限设置**

#### **修改前**
```java
// 设置其他默认值
user.setGender(wxUserInfo.getGender() != null ? wxUserInfo.getGender() : 1);
user.setPermissionLevel(3); // 默认为会员权限
user.setAmount(BigDecimal.ZERO);
user.setDiscount(BigDecimal.ONE);
```

#### **修改后**
```java
// 设置其他默认值
user.setGender(wxUserInfo.getGender() != null ? wxUserInfo.getGender() : 1);
user.setPermissionLevel(4); // 默认为游客权限，与短信注册保持一致
user.setAmount(BigDecimal.ZERO);
user.setDiscount(BigDecimal.ONE);
```

## 📊 **修改后的统一性**

### **新用户权限对比**
| 注册方式 | 权限级别 | 权限名称 | 权限描述 |
|---------|---------|---------|---------|
| 短信注册 | 4 | 游客 | 最低权限，只能进行基本查看 |
| 微信注册 | 4 | 游客 | 最低权限，只能进行基本查看 |

### **权限提升机制**
根据业务逻辑，游客在充值后会自动提升为会员：
```java
// 游客充值后自动提升为会员
if (user.getPermissionLevel() != null && user.getPermissionLevel() == 4) {
    user.setPermissionLevel(3); // 提升为会员
    // 生成会员号
}
```

## 🔄 **业务逻辑一致性**

### **统一的新用户体验**
1. **初始权限** - 所有新用户都是游客权限
2. **权限限制** - 只能查看公开信息和自己的数据
3. **权限提升** - 充值后自动提升为会员
4. **功能访问** - 根据权限级别控制功能访问

### **权限控制示例**
```java
// 游客和会员只能查看自己的记录
if (currentUser.getPermissionLevel() >= 3) {
    return currentUser.getPhone();
} else {
    // 员工和店长可以查看所有记录
    return null;
}
```
